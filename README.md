# LinkedIn 职位语言探测器

> 找工作已经够累了，还要点进去看是不是荷兰语/德语写的JD？不存在的。

## 这是啥？

一个 Chrome 扩展，帮你在 LinkedIn 搜索结果里直接看到每个职位是啥语言写的。

看到 **NL** 徽章？荷兰语职位，不会荷兰语的建议直接划走（或者赶紧学）。  
看到 **DE** 徽章？德语职位，祝你好运。  
啥徽章都没有？大概率是英语，点进去看看吧。

## 为啥要做这个？

因为本人在荷兰找工作被坑过无数次：
- 标题写得挺正常
- 点进去发现全文荷兰语
- 心态崩了

现在好了，不用点进去就知道是不是浪费时间。

## 怎么用？

### 安装（两种方法）

**方法一：开发者模式（推荐）**
1. 下载这个仓库
2. 打开 Chrome，输入 `chrome://extensions/`
3. 右上角打开「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹
6. 去 LinkedIn 搜工作，看徽章

**方法二：Chrome Web Store（等我有空了上传）**

还没传，先用方法一吧。

## 技术栈

- **语言检测**：ELD (Efficient Language Detector) - 就那个 264KB 的库，贼快
- **技术**：纯 JavaScript，Chrome Manifest V3
- **测试**：bun test（因为懒，不想配 Jest）

## 支持的语言

目前检测这些：
- 🇬🇧 英语 (en) - 默认不显示徽章，省得页面太乱
- 🇳🇱 荷兰语 (nl) - 红色徽章警告 ⚠️
- 🇩🇪 德语 (de) - 红色徽章警告 ⚠️
- 🇫🇷 法语 (fr)
- 🇪🇸 西班牙语 (es)
- 🇮🇹 意大利语 (it)
- 🇵🇹 葡萄牙语 (pt)
- 还有波兰语、瑞典语、丹麦语...

基本上覆盖欧洲主流国家了。

## 已知问题

1. **LinkedIn 经常改 DOM** - 如果某天突然不工作了，大概率是他们又改版了，提个 issue 我看看
2. **短描述检测不准** - 少于 50 个字符的基本跳过，不然误判率太高
3. **只会检测主要语言** - 如果是英法混合的 JD，可能只显示一个（通常是描述更多的那个）
